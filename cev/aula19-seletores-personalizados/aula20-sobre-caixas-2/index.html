Exemplo de caixa box-level

Parágrafos são exemplos de box-level, mas os links são exemplos de caixas inline-level

Display: block = display modelo de box-level, como h1, parágrafos| são uma caixa completa que utiliza toda a largura da tela, a não ser que seja configurado

Display: inline = display modelo inline-block, ocupam apenas a largura do conteúdo, permitindo conteúdos na mesma limha por padrão, como linls, botões, imagens, span

outline - como se fosse uma borda após a margin
