grow, basis ,shrink e order

Para um item ser flex (aumentar ou não em relação ao container), ele não pode ter tamanhos definidos, pois o flex irá respeitar a declaração do width

flex-grow inutiliza o justify-content

  • flex-grow - padrão 0 - quanto maior o flex-grow, maior o item esticará em relação aos outros,o item crescerá o maximo que conseguir. Os itens ocuparão todo o espaço

  • flex-basis - padrão auto - O = indica que todos os itens começarão com o a mesma base, ou seja, mesmo tamanho, se não aplicarmos isso, o item terá o tamanho do seu conteúdo

  • flex-shrink - padrão 1 - capacidade do elemento diminuir para caber no container

  • order - padrão - ordem de posicionamento - se adcionarmos uma order para um item, ele irá para a ultima posição - a ordem será do menor para o maior

  • flex - um atalho - flex:flex-grow, flex-shrink, flex-basis

  • align-self - aplicar no item flex, para alinhar somente ele isoladamente no eixo secundário
