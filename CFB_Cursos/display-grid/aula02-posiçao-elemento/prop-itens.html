Essas propriedades são aplicadas nos items


grid-column-start: define a coluna que o item irá começar


grid-column-end: define até que coluna o item irá, ele irá uma coluna antes;


grid-column: num/num; - podemos abreviar para esta formatação


grid-row-start: define a linha que o item irá começar


grid-row-end: define até que linha o item irá, ele irá uma linha antes;


grid-row: num/num; - podemos abreviar para esta formatação


grid-row ou column: num/span numero; - item irá expandir até o span indicado, e não até um antes


grid-area: junta todas as propriedades anteriores, na seguinte ordem:

grid-row-start / grid-column-start / grid-row-end / grid-column-end
